\# 📄 PDF Assistant — Document-Aware AI



An AI-powered web app that lets you upload a PDF and ask questions about it using Google Gemini.



\## 🔗 Live Demo

👉 https://pdf-assistant-yzx8xcvrh2jvambvnwnq2u.streamlit.app/



\## 🛠️ Built With

\- Python

\- Streamlit

\- Google Gemini API (gemini-2.0-flash-lite)

\- PyMuPDF



\## 🚀 Run Locally

```bash

git clone https://github.com/SKYDH08/pdf-assistant.git

cd pdf-assistant

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

streamlit run app.py

```



\## 📁 Project Structure

```

pdf-assistant/

├── app.py            # Main application

├── requirements.txt  # Dependencies

└── .gitignore        # Git ignore rules

```



\## ⚠️ Note

Add your own `GOOGLE\_API\_KEY` in a `.env` file to run locally.
